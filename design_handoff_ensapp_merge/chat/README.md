# Design — versione con Chat (assistente protocolli)

Metti qui i file di design relativi alla **chat / assistente** (mockup, export
Figma in HTML, immagini, note). Tienili separati dai mockup originali che stanno
in `../prototype/`.

Esempi di cosa mettere qui:
- `chat-screen.html` o `.png` — schermata della chat
- eventuali varianti / stati (vuoto, caricamento, risposta con fonti)
- note di stile specifiche della chat

Come caricarli da GitHub:
1. Apri questa cartella su GitHub.
2. **Add file → Upload files** → trascina i file → **Commit changes**.

Nota: questi file sono solo **riferimenti di design**, non vengono pubblicati
nell'app. L'app vera è in `src/` (la schermata chat è `src/routes/Assistant.svelte`).
